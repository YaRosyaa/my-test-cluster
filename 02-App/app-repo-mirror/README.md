# Устанавилваемое веб-приложение
Реальный репозиторий лежит на локальном GitLab. Тут - зеркало для демонстрации.

## Описание CI/CD
1. `lint`

    Проверка кода в линтерах *pylint* и *flake8*. Скачанные зависимости кешируются.
    
    Запускается при:
    - Коммите в *master*-векту
    - Наличии тега коммита
    - Merge реквесте

2. `build`

    Приложение собирается в Docker-образ, по возможности - используется кеш слоёв из последней сборки, полученной из registry. 
    
    Полученный образ сохраняется в артефакты.

    Запускается при:
    - Коммите в *master*-векту
    - Наличии тега коммита

3. `push`
    
    Итоговый образ пушится в registry с тегами:
    - `latest`
    - *тег коммита* - при наличии
    

    Запускается при:
    - Коммите в *master*-векту
    - Наличии тега коммита
