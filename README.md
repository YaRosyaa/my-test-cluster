# Этапы
```bash
├───00-Ansible           # Установка зависимостей и развёртывание кластера Kubernetes
│   └───roles
│       ├───prepare-host # Установка зависимостей + подготовка к развёртыванию
│       ├───first-master # Инициализация кластера на 1 control-plane ноде
│       └───add-host     # Добавление остальных хостов в кластере
├───01-Networking        # Обеспечение внешнего доступа в кластер
│   ├───01-cert-manager  # Создание CA и сертификатов
│   ├───02-metallb       # Настройка сервиса Load Balanceer от MetalLB
│   ├───03-gateway       # Создание Gateway с добавлением сертификата
│   ├───04-routes        # создание сервиса приложения и маршрутов к нему
│   └───charts
│       └───traefik      # values для Helm-чарта traefik
└───02-App               # Манифесты деплоя веб-приложения
```
# В процессе
1. Установка ArgoCD для GitOps развёртывания остальных компонентов кластера
2. Сбор и визуализация метрик кластера, нод, приложений: *VictoriaMetrics + Grafana*
3. Сбор логов нод и приложений: *FluentBit -> Kafka -> Vector -> OpenSearch*
